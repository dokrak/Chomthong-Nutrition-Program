วิธีการติดตั้งโปรแกรม
  • Copy folder hospital ไปไว้ใน root follder ของ website
  • ติดตั้งฐานข้อมูลด้วย hospital.sql ลงใน server โดยใช้ชื่อฐานข้อมูลเป็น hospital (ถ้าเปลี้ยนเป็นชื่ออื่นอาจมีปัญหาในการติดต่อฐานข้อมูลของบางไฟล์ได้)
  • การตั้งค่าเพื่อ connect database

    เข้าไปที่ hospital/tool/db.php

    เปิดไฟล์ db.php ด้วย web editor

    แก้ไขค่า $host ,$user , $password ในเครื่องหมาย " " ตาม IP server ของท่าน

    private static $host = "localhost";

    private static $user = "root";

    private static $password = "root";

    • ทำการ save และเริ่มใช้งานโปรแกรมได้
  • สร้าง link website เข้าสู่ โปรแกรมหลัก ด้วย ไฟล์ index.html
  • การเริ่มใช้โปรแกรม ให้ Log in เข้าสู่โปรแกรมด้วย

    User : admin

    Password : admin

    โดย

    -ค่า default ของ User กับ Password จะ set level ไว้ที่ level 3

    -Logo และชื่อโรงพยาบาลคือ รพ.จอมทอง

    -สามารถเข้าไปปรับเปลี่ยนข้อมูลพื้นฐาน เพิ่มรายชื่อของโรงพยาบาลท่านได้ โดยเข้าไปในส่วนของ ลงข้อมูลพื้นฐานของโรงพยาบาลที่จำเป็น

    -รายชื่อโรงพยาบาล(ต้องทำก่อนเป็นอันดับแรก) โดยเพิ่มรายชื่อ รพ.ของท่านเป็น รพ. แรก แล้วตามด้วย รพ.ในเครือข่าย(ถ้ามี)

    - ชื่อแพทย์

    -ชื่อผู้ใช้(user)

    -หน่วยงาน

    -รายชื่อสารอาหารที่มีใช้ในโรงพยาบาล (Enteral / Parenteral formular)

     

  • อาจจะเปลี่ยนแปลงหรือล้างข้อมูลเก่าในตาราง user , doctor ,department, en , pn และ hospital เพื่อเริ่มลงข้อมูลใหม่หมดเลยได้
  • วิธีการใส่ LOGO โรงพยาบาลของท่านและเครือข่าย
    • เปิด database hospital ==>เปิด table : hospital
    • พยายามให้ รพ.ท่านเป็นรายชื่อ แรกในตาราง ( id=1,hid=1) แล้วค่อยเพิ่มรายชื่อ รพ.เครือข่ายในภายหลัง
    • Save ภาพ logo ของ รพ. ของท่านใน folder hospital / img ==> ตั้งชื่อเป็ฯ hid.jpg (ชื่อต้องตรงกับ hid ของโรงพยาบาลท่าน เช่น รพ.จอมทอง hid=0 ==> save file logo เป็น 0.jpg)
    • เพิ่มรายชื่อ โรงพยาบาลอื่นๆ และ save file ภาพ Logo ของ รพ.นั้นๆ ใน folder hospital / img ==> ตั้งชื่อเป็ฯ hid.jpg (โดยชื่อของภาพต้องตรงกับ hid ของโรงพยาบาลนั้นๆ)
  • คำแนะนำการใช้โปรแกรมอื่นๆสามารถดูรายละเอียด และ Video Clip ได้ที่ link คำแนะนำการใช้โปรแกรม
  • การพัฒนาต่อยอดในอนาคต
    1. การ export data เป็น exel เพื่อนำไปใช้วิเคราะห์ นำเสนอ ปรับปรุงงานหรือวิจัยต่อไปได้
    2. การเชื่อมกับฐานข้อมูล HIS ของโรงพยาบาล เช่น hospXP หรือ อื่นๆ
